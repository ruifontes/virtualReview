# 虚拟浏览 #

* 作者：Rui Batista和NVDA Addon团队
* 下载 [稳定版][1]

此Addon允许NVDA用户在文本框中查看Window内容，类似于JAWS for
Windows的窗口虚拟化。但请注意，这只是为用户带来的便利，并不能取代NVDA出色的浏览模式和对象导航支持。

## 用法 ##

按下NVDA + control + w打开虚拟修订窗口。然后，像在任何其他文本内容中一样，只需导航显示的文本框。您可以按Escape关闭虚拟修订窗口。
